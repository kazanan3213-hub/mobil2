# Oturum İzni Rehberi (Android - Java)

Bu proje, yabancı uyruklu öğrencilerin oturum izni başvurusu için gereken evrakları ve online başvuru linkini gösteren basit bir Android uygulamasıdır.

## Android Studio ile açma
1. Android Studio > **Open** ile proje klasörünü seçin.
2. Gradle senkronizasyonunu tamamlayın.
3. Bir emülatör veya bağlı cihazda **Run** ile çalıştırın.

## İçerik
- Gerekli evrak listesi
- Online başvuru linki (tıklanabilir)
- Bilgilendirme notu

Kaynak link: https://e-ikamet.goc.gov.tr
